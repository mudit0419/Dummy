## ğŸš€ Getting Started

### Prerequisites
- Node.js v16+
- Python 3.9+
- MongoDB

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/Medicrypt.git
   cd Medicrypt
   ```

2. **Install frontend dependencies**
   ```bash
   cd frontend
   npm install
   ```

3. **Install backend dependencies**
   ```bash
   cd ../backend
   npm install
   ```

4. **Set up environment variables**
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

5. **Start the development servers**
   ```bash
   # In backend directory
   npm run dev
   
   # In a new terminal, in frontend directory
   npm start
   ```
